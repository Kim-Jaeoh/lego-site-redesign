## 📋 프로젝트

해당 프로젝트는 레고 사이트를 리디자인하여 Adobe XD로 와이어프레임 설계 후 코딩한 작업물입니다. <br>
간단한 회원가입 기능을 구현했으며, 기획부터 제작 과정을 거치면서 코딩하는 것에 재미를 붙일 수 있었습니다.

<br>

## 🛠️ 사용 기술

- `React`, `Css-Module`, `TypeScript`
- `Swiper`, `Antd`
- `Firebase`
- Deploy : `Netlify`

<br>

## ✨ 기능 및 특징

- `Adobe XD`로 와이어 프레임 설계 후 코딩
- `Firebase`를 사용해 회원가입/로그인 구현
- `Swiper`를 사용해 캐러셀 구현
